# Slides Tool Kit

## install

clone dis &&

```
npm install
```

## Run

```
npm start
```

This will open your slide show in a browser tab


## Edit

Point Your Favourite Editor(tm) at slides.md

edit dem

## Print
```
npm run print
```
This creates slides.pdf, which you can print
> Note: npm start needs to be running
